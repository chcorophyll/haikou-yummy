# 海口美食地图 (Haikou Yummy) 开发计划 🗺️🍜

## 🎯 核心目标与技术栈选型

*   **项目定位**：一款聚焦本地美食打卡、探店推荐的地图应用，兼顾游客的“必吃榜”与本地人的“私藏馆子”。
*   **平台支持**：Web 端 (响应式) + 微信小程序
*   **技术栈 (Tech Stack)**：
    *   **后端 API**：`Python` + `FastAPI` (轻量、高性能，非常适合快速开发和后期数据处理扩展)
    *   **数据库**：`MongoDB` (使用 MongoDB Atlas 免费云服务，文档型数据库极为契合位置数据和非结构化的店铺信息)
    *   **Web 前端**：`React` + `Tailwind CSS` (快速构建现代化、高颜值的界面)
    *   **小程序端**：`UniApp` (基于 Vue，一套代码编译到微信小程序，高效且成熟)
    *   **地图服务**：推荐使用 **高德地图 API** 或 **腾讯地图微信小程序插件**。

---

## 🗺️ 阶段一：MVP 构建 (最小可行性产品) - “让地图亮起来”

**此阶段核心目标：跑通前后端数据链路，用现有的不完整数据把核心地图功能做出来，验证想法。**

### 1. 基础设施搭建
*   **数据库**：注册 MongoDB Atlas 免费集群，创建 `restaurants` (店铺) 集合，设计基于 `GeoJSON` 的数据结构。
*   **后端 (FastAPI)**：
    *   初始化项目结构。
    *   连接 MongoDB。
    *   实现核心接口：获取店铺列表、根据地理位置进行附近搜索 (`$near` 查询)、获取单个店铺详情。
*   **数据清洗与导入**：
    *   写一个小脚本（Python），将你手上现有的不齐全新数据，清洗并导入到 MongoDB 中（缺失的字段先留空）。

### 2. 多端前端开发
*   **Web 端 (React + Tailwind)**：
    *   实现主页：左侧/上方为地图，右侧/下方为店铺列表卡片。
    *   实现检索：简单的关键词搜索或分类过滤。
    *   实现店铺详情侧边栏/弹窗。
*   **小程序端 (UniApp)**：
    *   使用原生 `map` 组件或集成地图插件。
    *   实现和 Web 端类似的逻辑：附近的美食地图标点、点击气泡查看简略信息、进入详情页。

***

## 🚀 阶段二：V1.5 数据补全与内容生态 - “让地图丰满起来”

**此阶段核心目标：解决数据不齐全的问题，引入爬虫技术，并建立管理后台。**

*   **爬虫与数据丰满 (Crawler)**：
    *   使用 Python (如 `BeautifulSoup`, `Playwright` 或 `Scrapy`) 编写轻量级爬虫脚本。
    *   针对现有数据，通过店名/地址匹配，从点评类网站或搜索殷勤抓取缺失信息 (如：营业时间、人均消费、联系方式、外部图片链接)。
*   **内容管理后台 (Admin Panel - React构建)**：
    *   开发一个简单的后台页面，方便你（作为超级管理员）审核爬虫抓取的数据，进行人工修改、下架或新增店铺。
*   **PGC 专题补充**：
    *   在地图中增加“专题/榜单”功能 (如：海大南门夜市必吃合集)，让你的人工精选内容脱颖而出。

***

## 🌟 阶段三：V2 用户互动与 UGC - “让地图活起来”

**此阶段核心目标：引入用户系统，让用户参与到地图的建设中来。**

*   **用户系统架构**：
    *   后端实现用户注册/登录 (小程序端可直接接入微信授权登录获取 OpenID)。
    *   使用 JWT 进行接口鉴权。
*   **UGC 赋能 (用户生产内容)**：
    *   **提交新店**：开放表单，允许用户在地图上长按选点，提交他们发现的“宝藏小店” (提交后进入 Admin 后台审核池)。
    *   **打卡与纠错**：用户可以标记已去过的店，或者提交纠错信息（如“已关门”、“营业时间有误”）。
    *   **收藏夹**：用户可以收藏喜欢的店铺，形成自己的美食心愿单。
*   **UI/UX 深度打磨**：
    *   引入优美的过渡动画，根据用户当前位置提供动态的距离提示和路线规划功能跳转。

## 🗂️ 数据库核心 Schema 构想 (MongoDB)

采用 MongoDB 特有的 `GeoJSON` 格式，方便直接利用空间索引：

```json
{
  "_id": "ObjectId",
  "name": "正宗吴日彪蒜香猪牛排",
  "category": ["小吃", "炸炸"],
  "location": {
    "type": "Point",
    "coordinates": [110.334001, 20.038481] // [经度, 纬度]
  },
  "address": "水巷口街...",
  "images": ["url1", "url2"],
  "price_per_person": 25,
  "opening_hours": "10:00-22:00",
  "rating": 4.8,
  "source": "pgc", // 数据来源: pgc / crawler / ugc
  "is_verified": true // 后台审核状态
}
```
