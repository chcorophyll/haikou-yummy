# 海口美食地图 (Haikou Yummy) 开发计划 🗺️🍜

## 🎯 核心目标与技术栈选型

*   **项目定位**：一款聚焦本地美食打卡、探店推荐的地图应用，兼顾游客的“必吃榜”与本地人的“私藏馆子”。
*   **平台支持**：Web 端 (响应式) + 微信小程序
*   **技术栈 (Tech Stack)**：
    *   **后端 API**：`Python` + `FastAPI` (轻量、高性能，非常适合快速开发和后期数据处理扩展)
    *   **数据库**：`MongoDB` (使用 MongoDB Atlas 免费云服务，文档型数据库极为契合位置数据和非结构化的店铺信息)
    *   **Web 前端**：`React` + `Tailwind CSS` (快速构建现代化、高颜值的界面)
    *   **小程序端**：`UniApp` (基于 Vue，一套代码编译到微信小程序，高效且成熟)
    *   **地图服务**：推荐使用 **高德地图 API** 或 **腾讯地图微信小程序插件**。
